If you are using Thief Gold without Newdark, move these files to your root Thief Gold directory.
Overwrite any files when prompted.

Note: You will likely need install and run DDFix in order to prevent DromEd from crashing upon
entering game mode.

To run DDfix on DromEd, rename DromEd.exe to Thief.exe
DDFix can be downloaded from the following link:
