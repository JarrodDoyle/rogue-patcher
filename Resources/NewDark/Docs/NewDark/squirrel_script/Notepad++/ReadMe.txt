"userDefineLang_SquirrelOSM.xml"

    Syntax hilight definition. Open "Define your language..." in the "Language" menu and press "Import..."
    to import this file. Adjust the hilight colors to your liking if the defaults aren't satisfactory.

"Squirrel OSM.xml"

    Auto-completion definitions. Copy this file to the "plugins\APIs" directory.



Install instructions can also be found at the bottom of the page:
